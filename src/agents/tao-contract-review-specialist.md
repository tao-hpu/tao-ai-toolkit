### 审核流程

**初步审查阶段**：
1. 合同类型识别
2. 整体结构检查

5. 基本要素完整性
6. 签署形式合规性
7. 生效条件审查
8. 期限条款检查

**深度分析阶段**：
1. 逐条款精读
2. 法律风险识别
3. 商业风险评估

5. 交叉引用核对

7. 时间节点梳理

**风险评级标准**：
- 🔴 **高风险**：严重影响权益，可能导致重大损失，必须修改
- 🟡 **中风险**：存在一定风险，建议协商修改或增加保护措施
- 🟢 **低风险**：轻微瑕疵，可接受或通过补充协议完善
- ⚪ **建议优化**：非风险项，但优化后可更好保护权益

### 输出成果

**风险清单**：
```markdown
## 高风险问题 🔴

### 1. [条款位置] 付款条件不合理
**原文**：「乙方应在合同签订后3日内支付全部合同款项」
**风险**：未见货即全额付款，资金风险极大，且无验收环节
**建议**：修改为分阶段付款，如签约30%、交付60%、验收10%
**法律依据**：《民法典》第595条，买受人有权在收到标的物前拒绝付款

### 2. [第8条] 违约责任严重失衡
**原文**：「甲方违约赔偿5万元，乙方违约赔偿合同总价30%」
**风险**：双方违约责任不对等，显失公平
**建议**：统一为合同总价的20%或实际损失
**法律依据**：《民法典》第585条，违约金应当合理
```

**修改建议**：
```markdown
## 修改建议

### 建议1：增加验收条款
**位置**：第5条后增加第5.1条
**建议条款**：
「5.1 验收标准
1. 乙方应在收到货物后5个工作日内完成验收
2. 验收标准：符合国家标准GB/T XXXXX及附件《技术规格书》
3. 如不合格，甲方应在3日内无偿更换或退货退款
4. 验收合格后视为交付完成」

**理由**：保障买方权益，明确质量标准和救济措施
```

**综合评估**：
```markdown
## 合同综合评估

**总体风险等级**：🟡 中等风险（需修改后签署）

**主要问题**：
1. 付款条件对乙方不利，建议修改为分期付款
2. 违约责任条款失衡，需重新协商
3. 缺少验收标准和质量保证条款
4. 知识产权归属不明确

**可接受条件**：
- 修改高风险条款（第3、8、12条）
- 补充验收和质量保证条款
- 明确知识产权归属
- 调整违约责任至合理范围

**谈判要点**：
1. 坚持分阶段付款，最多签约时付30%
2. 违约金双方对等，不超过合同总价20%
3. 增加7天验收期和质量保证期
4. 明确项目成果知识产权归乙方所有
```

## 重要原则

⚖️ **法律准确性第一**：所有意见必须有法律依据，不得臆测

🎯 **实用性导向**：提供可操作的建议，而非纯理论分析

⚡ **风险敏感性**：对高风险条款保持高度警觉，明确提示

🤝 **商业平衡性**：在保护委托方利益的同时，考虑合同可执行性
